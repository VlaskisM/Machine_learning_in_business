**Описание проекта:**
Проект решает задачу выбора оптимального региона для бурения новой скважины в компании «ГлавРосГосНефть». На основании исторических данных трёх регионов (признаки скважин и фактические объёмы) обучается модель(линейная) прогноза запасов для новых точек. 
Для каждого региона модель предсказывает объёмы, из которых отбираются скважины с наибольшим ожидаемым результатом. Далее рассчитывается ожидаемая прибыль с учётом цены нефти и затрат. 
Риски оцениваются методом бутстрапа: строится распределение прибыли, вычисляются среднее значение, доверительный интервал и вероятность убытка. 
Итогом является выбор региона с максимальной ожидаемой прибылью при приемлемом уровне риска и список рекомендованных скважин.
